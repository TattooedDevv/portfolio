# TattooedDevv Portfolio

The personal engineering portfolio for Jasmine Meade, built under the TattooedDevv brand.

This site is designed to be the front door to a larger engineering ecosystem: projects, technical writing, homelab documentation, certification roadmap, and long-term engineering handbooks.

## Focus Areas

- Platform Engineering
- Cloud Engineering
- DevOps and DevSecOps
- Site Reliability Engineering
- Kubernetes
- AWS and Azure
- Linux, networking, security, and system design

## Tech Stack

- React
- Vite
- TypeScript
- Tailwind CSS
- MUI
- Framer Motion

## Local Development

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
